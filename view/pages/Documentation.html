2. Overall Description

2.1 Product Description

The Game Changer web application, based on the porject ONT, has been developed for those that wish to visualize data about different board games and for those that wish to buy and bid for these products.

2.2 Product Functions

The web application manages family games. These games have associated data such as category, number of players, type, release year. The system offers support for buying and auctioning these products. The admin is offered with support for managing users. The application offers the option of exporting different statistics in formats such as CSV, WebP and PDF.

  • number of players : returns the games that can be played in the selected number of players
  • game type : returns the games that are in the selected game type
Auction :

  • bid -> make a bid : places the bid

4. System Features

4.1 Home page

Guests and users can read a little about the motivation behind the web app and navigate to all the other pages from this page.

4.2 Login page

Users of the app can login by entering their credentials and therefore access other functionalities of the app.

4.3 SignUp page

Guests can create an account and then use it to login into the application and access all the features the account existence provides.

4.4 Games page

Users can view the games based on different filters, add them to the cart and then proceed to purchase them. They can also modify the cart content.

4.5 Auction page

Users can view different collectible games, filter them and bid for getting into the possession of a desired game. Each game can be viewed in a different page where a short description , pictures and details about the selected game are provided.

4.6 Contact page

Guests and users can send a message containing feedback or any message that they wish to send to the authors of the application.

4.7 Admin page

The admin can login into the app by entering his credentials and then be redirected to the admin page where he can manage users, products and other aspects of the website.

4.8 Terms and Conditions page

Guests and users can read the terms and conditions of using the website by navigating to this page.

4.9 How to buy

Guests and users can find out more information about the purchasing process by visiting this page.
